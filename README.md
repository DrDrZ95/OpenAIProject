# Example Framework Project

This repository contains a minimal example for a Python project. The `app.py`
script prints `Hello, world!` when executed.

It also includes a minimal **Blazor** web application located in the
`BlazorAppNet8` directory. The solution is generated using the .NET SDK and targets
`.NET 8.0`.

## Running

```bash
python app.py
```

## Testing

Run the tests using `pytest`:

```bash
pytest
```
